# Security Policy

## Supported Versions

Currently only the first final release is supported.

| Version | Supported          |
| ---: | ------------------ |
| >= 1.0.2   | :white_check_mark: |
| <= 1.0.1  | :x:                |
| 0.x.x     | :x:                |

## Reporting a Vulnerability

Please report any found vulnerabilities to this repositories' owner.
